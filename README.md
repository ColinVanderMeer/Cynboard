# Cynboard
This is my submission for Hack Club Blueprint / sorta Hackpad V5. Thanks Cyaopad for serving as my go to board to look at when I had a problem.

## Features
- Hidden F13-F24 Keys for extra hotkeys in programs and games
- Midi layer for programs that don't accept anything but Midi
